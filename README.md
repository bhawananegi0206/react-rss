# react-rss

## Installation(Install packages)
```
npm install
```

### Compiles and Start a local server with:
```
npm start
```

### Compiles and minifies for production
```
npm run build
```

### Format all files
```
npm run format
```


### Local rss feed url to test

http://localhost:8080//src/rssFeed/rssFeedSample.xml
